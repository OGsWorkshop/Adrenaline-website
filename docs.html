Introduction

A short guide to getting Adrenaline installed and finding your way around.

These docs focus on the current client and the parts of the workflow that are supported today. We would rather keep this page useful than fill it with promises.

Before you start

You will need a Windows 10 or Windows 11 computer and a current Adrenaline download. Keep your license email nearby if you purchased a key.

Installation

The setup is intentionally small.

  1. Download the latest client from the home page.
  2. Open the downloaded file and follow the prompts.
  3. Launch Adrenaline and sign in or enter your key when asked.
Note: Only download files linked from the official site or community. If a mirror asks for your password, close it.

Workspace

The workspace keeps the editor and output close together.

Use the editor for your script, the output area for messages, and the run control when you are ready to test. Start with a small script so it is easy to spot where something goes wrong.

Keep things tidy

Use clear file names and save copies of scripts you are still working on. The client is not a replacement for a proper backup.

Scripts

A few habits make script work less frustrating.

  • Test one change at a time.
  • Read the output when a script stops unexpectedly.
  • Do not paste code you do not understand into a live session.

Updates

Check the official download page when a new client build is announced.

Older builds may stop behaving as expected after platform changes. If you update, keep the previous installer until you have confirmed that the new build works for you.

Troubleshooting

Start with the boring fixes first.

  • Restart the client and the game.
  • Make sure you are running the current download.
  • Check that your key has not expired.
  • Look through the community announcements for known issues.

If the issue remains, send a clear description and the relevant output to the Discord support channel.

FAQ

A few answers before you reach out.

How long does a key last?

Premium keys last one week. Enterprise keys last one month. The duration is shown again before you pay.

Where do I get my key?

It is sent to the email used during checkout after payment is confirmed.

Where can I ask for help?

Join the Discord community and include what you tried, what happened, and any useful output.
