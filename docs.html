Installation

Follow these steps to get Adrenaline up and running on your system.

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • 4GB RAM minimum (8GB recommended)
  • 500MB free disk space
  • Stable internet connection

Step-by-Step Guide

  1. Download the latest version of Adrenaline from our pricing page.
  2. Temporarily disable Windows Defender or add an exception for the Adrenaline folder.
  3. Extract the downloaded ZIP file to a folder of your choice.
  4. Run Adrenaline.exe as Administrator.
  5. If you purchased Premium, enter your license key when prompted.
  6. Launch Roblox and click "Inject" in the Adrenaline window.
