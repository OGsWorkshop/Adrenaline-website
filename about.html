Adrenaline is a small, focused scripting client. We care more about a dependable workspace and honest documentation than a long list of promises.

What we are working on

The project is built around a simple idea: make the tools easy to find, make the license terms easy to understand, and keep the client comfortable to use. The site, documentation, and community are maintained as one place to start.

A note on support

If something is not working, send a clear description through the official community server. Include the client version and the relevant output, but never share passwords, private keys, or files you do not own.

Find the official links

Only trust downloads and announcements linked from this website or the official Adrenaline Discord server. If somebody claims to be staff and sends you a different link, check with the community before opening it.
